# vehicle-monitoring
Vehicle monitoring using YOLOv8, DeepFace, OpenV and TKinter


## How to run:
Step 1:
clone/fdownload the repository the repository

Step 2: 
Open the terminal in root directory 

Step 3: 
````
pip install requirements.txt
````
Step 4:
````
python vehicle-monitoring.py
````
